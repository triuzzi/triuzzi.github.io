Tab Shortcuts

Missing keyboard shortcuts for Chrome.
Tab management and GitHub productivity.

Install from Chrome Web Store

Features

Pin Tab

Ctrl+Shift+P

Toggle the pinned state of the current tab.

Duplicate Tab

Ctrl+Shift+D

Create a copy of the current tab.

Prevent Tab Close

Configure in settings

Prevent the current tab from being accidentally closed. Chrome will ask for confirmation before closing.

Collapse Translation Files

Configure in settings

On GitHub PRs, collapse all translation files (en.json, de.json, ...) to focus on code changes.

Convert Relative Time

Configure in settings

On GitHub, convert "2 hours ago" to absolute timestamps like "10 September 2025, 10:15".

Details

No data collected. No network requests. Runs entirely in your browser. Open source under MIT license.
